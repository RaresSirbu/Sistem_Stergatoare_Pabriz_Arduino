Acest proiect prezintă dezvoltarea unui sistem de tip ștergătoare adaptive pentru parbriz, care controlează două servo-motoare pentru a simula funcționarea reală a ștergătoarelor. Sistemul ajustează automat viteza de operare în funcție de intensitatea ploii detectate de un senzor, oferind o reacție fluidă și adaptivă la condițiile de mediu. De asemenea, este inclus un buton care permite utilizatorului să efectueze un singur swipe manual, oferind control direct.

Proiectul reprezintă un pas important în explorarea tehnologiilor de asistență a șoferului și a sistemelor automate de control, integrând elemente de electronică, senzori și programare într-un sistem funcțional și coerent. Scopul principal a fost realizarea unei simulări realiste a comportamentului ștergătoarelor adaptive.
